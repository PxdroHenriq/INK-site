# INK — Plataforma de Portfólios Criativos

## Sobre o Projeto  
O **INK** é uma plataforma web inspirada em sites de portfólios profissionais, permitindo que usuários explorem projetos criativos, filtrem categorias e visualizem detalhes individuais de cada trabalho.

O sistema inclui:

- Tela de **login simples**
- Página principal com **grid de projetos dinâmicos**
- Filtros por categoria
- Página de detalhes do projeto
- Interface moderna e responsiva

O objetivo do projeto é demonstrar princípios de **JavaScript dinâmico**, manipulação do DOM, estrutura modular e design inspirado em plataformas como Behance e Dribbble.

## Funcionalidades

### Login  
- Validação simples no front-end  
- Exibe a página principal somente após credenciais válidas

### Exibição de Projetos  
- Renderização dinâmica dos cards via JavaScript  
- Função `mostrarProjetos()` atualiza automaticamente o grid  

### Filtro por Categorias  
- Botões para categorias (UI/UX, Branding, Ilustração etc)  
- Filtragem instantânea sem reload  

### Página de Detalhes  
- Exibe descrição completa e imagem grande  
- Botão para retornar ao grid

### Estilo  
- Layout moderno e responsivo  
- Uso de Flexbox + CSS Grid  
- Animações suaves e efeitos de hover
